# ClawdMate · Design Specification
**版本**: v2.0 · 日杂像素风 × 橙色主系 × 明暗撞色**

---

## 一、设计定调 Design Direction

**风格关键词**: 日杂排版 / 像素手工感 / 轻复古 / 温暖克制

这套设计的核心张力来自于：**暖橙（活泼）× 格纹底（克制）× 深色进行中卡片（戏剧感）**。
整体像一本精心排版的工作手帐，轻量但不廉价，有质感但不沉重。

---

## 二、颜色系统 Color System

| `--o4`     | `#F8E4D0` | 极浅橙：卡片头背景、悬浮态          |
| `--o5`     | `#FDF5EE` | 奶橙白：卡片底色、侧边栏底色         |

### 撞色 — 钢蓝灰（冷、沉）

| 变量         | 色值      | 用途                                  |
|------------|---------|---------------------------------------|
| `--steel`  | `#3D4F58` | 侧边栏背景、进行中卡片背景（最暗板块）       |

> **撞色逻辑**：钢蓝灰与暖橙在色相环上相距约150°，属于近似互补但不完全对立的关系。两者饱和度都偏低，不会产生视觉冲突，反而形成「暖工作台 vs 冷任务舱」的空间感。

## 三、明暗分层 Light / Dark Zoning

整个界面从左到右形成 **暗→中→亮** 的明度渐变，制造层次感和空间感：

```
┌──────────────┬───────────────────────────────┬─────────────┐
│  侧边栏       │  主内容区                        │  右侧面板    │
│  深钢蓝灰     │  暖米白 + 斜纹底                  │  奶白最亮    │
│  #3D4F58     │  #F9F4EE                        │  #FDFAF5   │
│  【最暗】     │         【中】                   │  【最亮】   │
└──────────────┴───────────────────────────────┴─────────────┘
```

主内容区内部还有第二层分层：
- **添加任务卡**：`#FDFAF5` 奶白，明亮轻盈 → 代表「待激活」
- **进行中卡**：`#2E3F48` 深钢蓝 → 代表「沉浸专注」

这两张卡的对比度是页面的视觉高潮。

## 四、背景纹理 Background Texture

主内容区用纯 CSS 斜纹替代格子，更细腻不喧宾夺主：

```css
background-image: repeating-linear-gradient(
  -45deg,
  transparent,
  transparent 8px,
  rgba(200,170,140,0.06) 8px,
  rgba(200,170,140,0.06) 9px
);
```

- 角度：`-45°` 斜纹，比正格子更有手工感
- 透明度：极低（6%），不干扰内容阅读
- 底色：`#F9F4EE` 暖米白，斜纹隐约可见

## 五、像素感规范 Pixel Aesthetic Rules

### 5.1 像素阴影（核心技法）
所有卡片和主按钮使用**实色偏移阴影**，而非高斯模糊阴影：

```css
/* 添加任务卡 */
box-shadow: 3px 3px 0 var(--bg3);

/* 进行中卡 */
box-shadow: 3px 3px 0 var(--o3);

/* 开始任务按钮 */
box-shadow: 3px 3px 0 var(--o3);

/* 生成日报按钮 */
box-shadow: 2px 2px 0 var(--steel3);
```

规律：X/Y 偏移相等，颜色比主色浅一级，无模糊半径。

### 5.2 像素进度条
15格独立方块，每格有独立描边：

```css
.pp { 
  flex: 1; height: 7px;
  border: 1.5px solid var(--bg3); 
  background: rgba(255,255,255,0.06);  /* 在深色卡上 */
}
.pp.on  { background: var(--o);  border-color: var(--o2); }   /* 已完成格 */
.pp.tip { background: var(--o3); border-color: var(--o3); }   /* 当前格（渐出效果）*/
```

### 5.3 像素装饰线
标题下方用多段不等宽色块模拟印刷色带：

```html
<div style="display:flex; gap:3px; margin-top:10px;">
  <span style="width:28px; height:3px; background:var(--o)"></span>
  <span style="width:14px; height:3px; background:var(--o2)"></span>
  <span style="width:44px; height:3px; background:var(--o3)"></span>
  <span style="width:18px; height:3px; background:var(--o4)"></span>
</div>
```

### 5.4 Logo 方块
```css
.logo-sq {
  width: 32px; height: 32px;
  background: var(--o);
  border: 2px solid var(--o2);
  box-shadow: 2px 2px 0 rgba(0,0,0,0.25);  /* 在深色背景上加深阴影 */
}
```

## 六、组件规范 Component Specs

### 6.1 侧边栏 Sidebar
- 背景：`#3D4F58`（钢蓝灰，全页最暗）
- Logo区：无额外背景，直接在深色上
- 导航选中：`border-left: 2px solid var(--o)` + `background: rgba(255,255,255,0.07)`
- 导航文字：未选中 `--steel3`，选中 `#F0E8E0`
- 像素方块指示器：未选中空心，选中填橙色
- 召唤按钮：橙色实色，`box-shadow: 2px 2px 0 rgba(0,0,0,0.2)`

### 6.2 添加任务卡
- 背景：`#FDFAF5`（奶白，最亮）
- 边框：`1.5px solid var(--bg3)`，阴影 `3px 3px 0 var(--bg3)`（暖灰色）
- 卡头：`--bg2` 暖米色底
- 模式标签（极简模式）：`--steel4` 底 + `--steel2` 文字，与橙色形成撞色点缀
- 日期chip：同上，`JetBrains Mono` 等宽字体

### 6.3 进行中卡（视觉重心）
- 背景：`#2E3F48`（深钢蓝，全页最暗的卡片）
- 卡头：`--steel` + 橙点动画 + 橙色「当前聚焦」tag
- 任务名：`Noto Serif SC`，颜色 `#F0E8E0`（暖白）
- 计时数字：`JetBrains Mono 700`，37px，颜色 `#F0E8E0`
- 计时框：`background: rgba(0,0,0,0.2)` 内嵌感
- 三按钮：
  - 暂停：半透明白底，`--steel3` 文字
  - 完成：`rgba(217,120,64,0.12)` 橙底，橙色文字
  - 取消：`--steel2` 描边，`--steel3` 文字（比暂停更淡，降低存在感）

### 6.4 右侧统计面板
- 背景：`#FDFAF5`（最亮，与侧边栏形成最大反差）
- 统计卡片：`--bg2` 底，`2px 2px 0 --bg3` 像素阴影
- 生成日报按钮：`--steel4` 底 + `--steel2` 文字（用钢蓝撞色，不用橙色，避免重复）

## 七、字体规范 Typography

| 用途 | 字体 | 大小 | 字重 | 颜色 |
|------|------|------|------|------|
| 应用名 ClawdMate | Noto Serif SC | 14.5px | 500 | `#F0E8E0`（在深色上）|
| 页面欢迎语 | Noto Serif SC | 23px | 500 | `--ink` |
| 任务名称 | Noto Serif SC | 19px | 500 | `#F0E8E0`（在深色卡）|
| 正文 / 导航 | Noto Sans SC | 12px | 400 | `--i2` |
| 辅助文字 | Noto Sans SC | 9–10px | 400 | `--i3` / `--steel3` |
| 计时器数字 | JetBrains Mono | 37px | 700 | `#F0E8E0` |
| 进度比例 | JetBrains Mono | 13px | 700 | `--o` |
| 日期chip | JetBrains Mono | 10px | 500 | `--steel2` |
| 统计数字 | JetBrains Mono | 20px | 700 | `--ink` / `--o` |

**字体搭配逻辑**：衬线（Noto Serif SC）承担品牌感和标题层次；等宽（JetBrains Mono）强化数字的精确感和工具属性；无衬线（Noto Sans SC）处理所有正文保持阅读舒适。

## 八、间距与圆角 Spacing & Radius

- 圆角：**全部使用 1–2px 极小圆角**（接近直角），保持像素手工感，禁止大圆角
- 主内容区 padding：`26px 20px`
- 卡片内 padding：`15px`
- 导航项 padding：`8px 10px`
- 组件间 gap：`14px`（双栏布局）/ `12px`（卡内元素）
- 描边宽度：`1.5px`（一般描边）/ `2px`（强调描边，如进行中卡、召唤按钮）

## 十、设计意图总结 Design Intent

| 设计选择 | 意图 |
|---------|------|
| 侧边栏用深钢蓝 | 导航永远是最暗的，形成视觉锚点，不抢主内容 |
| 进行中卡用深色 | 强化「专注模式」氛围，与浅色添加卡形成戏剧对比 |
| 右面板最亮 | 统计信息是「奖励区」，明亮感带来成就感 |
| 撞色用钢蓝而非绿/紫 | 与橙色协调但不甜腻，保持工具属性的理性感 |
| 格纹换斜纹 | 斜纹更轻盈，格纹过于整齐容易显得拘谨 |
| 极小圆角 | 维持像素/印刷质感，避免现代 App 的圆润感 |
| 衬线字体做标题 | 增加日杂气质，区别于普通效率工具 |
